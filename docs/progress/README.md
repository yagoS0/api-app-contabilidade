# Progress Log

Use este diretório para registrar a progressão das mudanças do projeto.

Ordem de leitura sugerida para uma nova IA:
1. `docs/progress/current.md`
2. O arquivo datado mais recente em `docs/progress/`

Regras simples:
- Atualize `current.md` ao final de cada sessão.
- Crie um novo arquivo datado quando houver uma entrega relevante.
- Registre sempre:
  - o que foi alterado
  - onde foi alterado
  - como foi feito
  - pendências e próximos passos

Comando útil:
- `npm run progress:update -- --title "..." --change "..." --backend "file1|file2" --frontend "file3" --how "passo1|passo2" --notes "obs1|obs2"`
